Ctrl + I 字体倾斜
Ctrl + U 下划线

Ctrl + T 创建表格
Ctrl + K 创建超链接
Ctrl + F 搜索
Ctrl + H 搜索并替换
Ctrl + L 选中某句话   
Ctrl + D 选中某个单词
